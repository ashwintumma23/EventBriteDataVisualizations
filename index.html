Welcome to Data Visualizations for Eventbrite.

This page contains the links to the running demos of all the visualizations that were developed as a part of this project. Click on the links below to have a look at the different types of data visualizations.

  1. Heatmap of Events throughout the Globe using Google Maps API v3
  2. Bubble Chart of different formats on Eventbrite
  3. Distribution of events according to different states in United States of America (using D3.js)
  4. Calendar Heatmap of events according to the start time of the event (Using D3.js)

    5. Programming Languages and Technologies

    Python, Google Maps API v3, D3.js, jQuery
